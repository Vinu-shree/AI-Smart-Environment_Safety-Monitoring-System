# AI Smart Environment & Safety Monitoring System

## Overview
This project is an AI-based smart monitoring system developed using Python simulation and cloud dashboard integration.

It monitors environmental conditions such as:

- Temperature
- Gas Detection
- Light Intensity
- Motion Detection

The system calculates a **Risk Score** and automatically shows:

- SAFE
- WARNING
- DANGER

## Features

✅ Real-time sensor simulation  
✅ Cloud dashboard monitoring  
✅ Risk score calculation  
✅ Smart alert system  
✅ Email notification support  
✅ Future hardware implementation planned
